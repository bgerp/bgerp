Клавишни комбинации:
[#img1#] Ctrl + A - Избор на артикул
[#img2#] Ctrl + S - Подробни настройки на избрания артикул
[#img3#] Ctrl + Z - Плащане по бележката
[#img4#] Ctrl + K - Избор на контрагент и прехвърляне
[#img5#] Ctrl + Е - Добяване на текст към артикула
[#img6#] Ctrl + B - Преглед на бележките
[#img7#] F2 - Подробна информация за избрания елемент
[#img8#] Ctrl + P - Печат на бележката
[#img9#] Ctrl + V - Отваряне на виртуална клавиатура
[#img10#] Ctrl + O - Оттегляне на бележка
[#img11#] F1 - Помощна информация
[#img12#] Ctrl + X - Изход терминала
Ctrl + DEL - Изтриване на избрания ред от бележката
Съкратени формати за редактиране на текущ ред:
20* - Промяна на количество
*1.54 - Промяна на цената
5% - Задаване на отстъпка
